# SIFT 128维描述子生成

## 适用条件
- 输入必须是已计算出主方向（key.w）、尺度（key.z）和坐标的SIFT特征点
- 特征点周围的16×16像素邻域必须完全位于图像边界内
- 必须提供window_factor参数（通常默认为3.0）

## 执行流程

### 1. 坐标变换以实现旋转不变性
- 将16×16邻域划分为4×4个子块（每个子块4×4像素）
- 对每个子块中心(ix-1.5, iy-1.5)，应用旋转与缩放变换：
  ```
  pt.x = c * spt * (ix - 1.5) - s * spt * (iy - 1.5) + key.x
  pt.y = c * spt * (iy - 1.5) + s * spt * (ix - 1.5) + key.y
  ```
  其中：
  - `c = cos(key.w)`, `s = sin(key.w)`
  - `spt = |key.z| * window_factor`

### 2. 遍历子块采样窗口
- 对每个子块内所有像素(x, y)，计算其相对于旋转后子块中心的归一化坐标(nx, ny)
- 若`|nx| ≥ 1`或`|ny| ≥ 1`，则跳过该像素（超出双线性插值有效范围）

### 3. 梯度信息加权累加
- 获取当前像素的梯度幅值(cc.x)和方向(cc.y)
- 计算三重权重：
  - 高斯权重：`ww = exp(-0.125 * (dnx² + dny²))`
  - 双线性权重：`wx = 1 - |nx|`, `wy = 1 - |ny|`
  - 总权重：`weight = ww * wx * wy * cc.x`
- 计算相对于主方向的角度：`theta = (key.w - cc.y) * rpi`（其中`rpi = 4/π`）
- 将theta映射到0–7的8-bin直方图，使用线性插值将权重分配至相邻bin

### 4. 输出描述子
- 每个4×4子块生成一个8维向量，共4×4×8 = 128维
- 描述子最终存储为32个float4元素（即两个float4数组des[0–3]和des[4–7]按块拼接）

> **注意**：生成后需对整个128维向量进行L2正规化，并可选地进行阈值截断（如>0.2设为0.2）后再正规化，以增强光照鲁棒性。
