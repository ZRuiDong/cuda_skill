# 预定义结构元素模板

## 十字形结构元素（5×5中心对称）
有效坐标偏移（相对于中心）：
```
(0, -2), (0, -1), (0, 0), (0, 1), (0, 2),
(-2, 0), (-1, 0), (1, 0), (2, 0)
```
共9点。

## 圆盘形结构元素（近似5×5圆盘）
有效坐标偏移：
```
(-2,-1), (-2,0), (-2,1),
(-1,-2), (-1,-1), (-1,0), (-1,1), (-1,2),
(0,-2), (0,-1), (0,0), (0,1), (0,2),
(1,-2), (1,-1), (1,0), (1,1), (1,2),
(2,-1), (2,0), (2,1)
```
实际常用15点子集（如表8.4所示）。

## 邻域采样实现示例
```cpp
// 示例：十字形15点（含扩展）
int offsets[15][2] = {
  {0, -2}, {0, -1}, {0, 0}, {0, 1}, {0, 2},
  {-2, 0}, {-1, 0}, {1, 0}, {2, 0},
  {-1, -1}, {-1, 1}, {1, -1}, {1, 1},
  {-2, -1}, {2, 1} // 可根据实际结构调整
};
for (int i = 0; i < 15; ++i) {
  int nx = xIdx + offsets[i][0];
  int ny = yIdx + offsets[i][1];
  ucDomain[i] = tex2D(texRef, nx, ny);
}
```

> 注意：所有坐标需在图像范围内，越界处理应在核函数中显式实现（如返回255或复制边界）。
