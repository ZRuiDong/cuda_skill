# CUDA 高帽滤波三阶段实现

## 适用条件
- 输入图像为 uint8 灰度图
- 结构元素必须为圆盘形
- 已绑定 2D 纹理内存（texRef）
- 场景目标：校正不均匀背景并增强亮细节

## 执行流程

### 第一阶段：圆盘腐蚀
- 每个 CUDA 线程处理一个像素 (xIdx, yIdx)
- 遍历以该像素为中心、半径为 `uiRadius` 的圆盘区域
- 利用四象限对称性访问邻域：(±i, ±j)，其中 √(i² + j²) ≤ uiRadius
- 取所有有效邻域像素的最小值作为腐蚀输出

### 第二阶段：圆盘膨胀
- 对第一阶段的腐蚀结果执行膨胀操作
- 同样遍历圆盘邻域，但取最大值
- 注意修正代码笔误（如变量名 yIndex 应为 yIdx）

### 第三阶段：原图减开运算结果
- 开运算 = 腐蚀后膨胀
- 对每个像素计算：`tophat_result = max(0, original_pixel - opening_pixel)`
- 使用纹理引用 `texRef` 高效读取开运算结果
- 输出为高帽滤波图像，突出原始图像中的亮细节

## 性能提示
- 大半径 (`uiRadius`) 会显著增加计算量，导致性能下降
- 所有阶段均依赖纹理内存优化全局内存访问
- 线程网格应覆盖整幅图像尺寸

> **触发条件**：当需加速 `imtophat` 且结构元素为圆盘形时，立即采用此三阶段 CUDA 流程。
