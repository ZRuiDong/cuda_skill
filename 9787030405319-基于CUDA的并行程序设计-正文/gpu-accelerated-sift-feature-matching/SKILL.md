# GPU加速的SIFT特征匹配

## 适用条件
- 输入为两组L2正规化的SIFT特征描述子（每行模长为1）
- 已设定合理的`distmax`（最大允许匹配距离，单位：弧度）和`ratiomax`（NNDR阈值，通常为0.7–0.8）
- 每个查询特征至少存在两个近邻（满足NNDR计算要求）

## 执行步骤

1. **构建相关系数矩阵**
   - 将图像I的m个描述子组织为矩阵A（m×128）
   - 将图像J的n个描述子组织为矩阵B（n×128）
   - 在GPU上计算点积矩阵 C = A × Bᵀ（结果为m×n）

2. **执行行方向匹配（图像I → 图像J）**
   - 对C的每一行，并行归约找出最大点积`dotmax`和次大点积`dotnxt`及其索引
   - 将点积转换为角度距离：`dist = acos(min(dotmax / 262144.0, 1.0))`
   - 应用双重阈值检验：
     - `dist < distmax`（绝对距离约束）
     - `dist < distn * ratiomax`（NNDR约束，`distn`为次近距离）

3. **执行列方向匹配（图像J → 图像I）**
   - 对C的每一列重复类似归约与阈值判断，实现反向验证

4. **输出最终匹配结果**
   - 仅保留通过双向阈值检验的匹配对
   - 返回匹配索引数组；无效匹配位置标记为-1

> **注意**：本技能不适用于未正规化的描述子，且依赖GPU并行计算能力以实现高效匹配。
