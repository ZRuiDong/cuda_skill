# 简单任务图判定

## 何时使用
- 给定一个有向无环任务图（DAG）G
- 需要判断该图是否属于“简单任务图”以指导调度或优化
- 已知所有从初始任务到最终任务的路径（链）

## 执行步骤
1. **提取所有链**：找出任务图 G 中所有从初始任务到最终任务的路径，记为 C₁, C₂, ..., Cₘ。
2. **构建链多项式**：对每条链 Cᵢ = (Tᵢ₁, Tᵢ₂, ..., Tᵢⱼ)，用变量乘积 Xᵢ = xᵢ₁·xᵢ₂·...·xᵢⱼ 表示，其中 xₖ 对应任务 Tₖ。
3. **构造总多项式**：将所有链的多项式相加，得到 P(G) = X₁ + X₂ + ... + Xₘ。
4. **执行因式分解判定**：
   - 若 P(G) 可通过代数方法（如提取公因子、分组分解）化简为更紧凑形式（如乘积形式），则 G 是**简单任务图**，返回 true。
   - 若 P(G) 无法因式分解（即为不可约多项式），则 G 是**非简单任务图**，返回 false。

> 注意：仅适用于静态、确定性的有向无环图（DAG），不适用于动态或不确定任务图。
