# DoG极值点三步检测法

## 适用条件
- 已成功构建DoG（Difference of Gaussians）图像金字塔
- 已设定关键参数：`dog_threshold0`（对比度粗筛阈值）和`edge_threshold`（边缘抑制阈值）
- 当前处理的像素位于DoG图像内部区域（即行号>0、列号>0，且不在边界）

## 执行步骤

1. **8邻域对比与对比度阈值检查**
   - 将当前像素与其所在DoG层的8个空间邻域像素比较
   - 若其绝对值 |v| ≤ `dog_threshold0`，则直接剔除（非显著点）

2. **跨尺度18邻域极值验证**
   - 将该像素与相邻两个尺度DoG图像中各9个邻域像素（共18个）进行比较
   - 仅当其为所有26个邻域（8+9+9）中的局部极大值或极小值时，保留为候选点

3. **边缘效应抑制**
   - 计算Hessian矩阵二阶偏导数：`fxx`, `fyy`, `fxy`
   - 计算行列式 `temp1 = fxx * fyy - fxy²` 和迹平方 `temp2 = (fxx + fyy)²`
   - 若 `temp1 ≤ 0` 或 `temp2 > edge_threshold * temp1`，则判定为边缘响应点并剔除

## 输出结果
- 返回极值类型：`1.0`（极大值）、`-1.0`（极小值）或 `0.0`（非极值）
- 同时输出亚像素级偏移量 `(dx, dy, ds)`（用于后续精确定位）

> **性能提示**：第一步可剔除约95%的像素，显著提升GPU并行效率，适用于大规模图像处理场景。
