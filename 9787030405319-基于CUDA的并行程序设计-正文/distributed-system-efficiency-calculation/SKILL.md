# 分布式系统运行效能计算

## 何时使用
- 需要评估同一问题规模（如固定N_max=7）下，n节点系统相对于2节点基准的并行效率。
- 已知2节点基准运行总时间（T_2）和n节点实际运行总时间（T_n），且n ≥ 2。
- 系统运行于同构硬件环境，且适用于特征值求解器或绿色计算评估场景。

## 如何执行
1. 获取2节点配置下的系统总运行时间 T_2（单位：秒）。
2. 获取目标n节点配置（n ≥ 2）下的系统总运行时间 T_n（单位：秒）。
3. 应用公式：**系统运行效能(n) = (2 × T_2) / (n × T_n)**。
4. 将结果乘以100，以百分比形式表示效能（例如：1.0 → 100%，0.95 → 95%）。
5. 解读结果：值越接近100%，表示资源利用效率越高，绿色计算效果越好。

> 注意：此方法假设2节点为最小可行配置，且仅适用于同构硬件环境下的横向比较。
