# 性能数据参考

## 典型硬件参数
- Carver节点内存：24 GB（实际可用约15 GB）
- 单数据节点SSD理论带宽：2 GB/s

## 性能表现示例（来自图9.14与表9.6）
- 核外计算实测带宽范围：81–129 MB/s
- 36节点集群实测总带宽：≈18.5 GB/s（接近8个数据节点 × 2 GB/s = 16 GB/s上限）

## 成本计算公式
cost_total = i_total × t_total
其中：
- i_total：CPU处理核总数
- t_total：任务总耗时（受I/O瓶颈显著影响）

## 优化效果对比（表9.8摘要）
- 9节点配置在某些场景下比36节点更高效，因减少I/O竞争
